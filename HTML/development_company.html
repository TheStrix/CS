Google Android

What is Android?

Android is developed by Google Inc. First Android was released on September 23, 2008. Android is an operation system which is based on Linux-OS. It's a mobile OS. At the moment, 1 billion users have Android OS on their phones. Android is the most used mobile OS. The latest version of Android is Android 5.1(lollipop).

Development

Android OS is a Open Source project. It's source code can be found here. The Android OS can be built using Linux OS(Any distro) or Mac OS. OS Setup tutorial can be found here. The source building tools is mainly written in C++ and C and system apps are written in XML and Java. The whole Android source code is of about 20GB and requires about 40GB of free space for compiling from source.

Releases:

The first Android version (Android Alpha 1.0) was released on September 2008. The latest release is Android Lollipop 5.1(API 22). The Android codename are released on alphabatical order; just linux Linux OS. Till date, Android Lollipop is the biggest update with new looks called Material Design.

Building

To port Android to a new device, we need to work on tree major components.

  1. Device tree

    2. The major configuration part for a device is done here. As the name itself points, it is device-specific; means, every device have it's own config setup. Many examples of device tree can be found on Google's Android source git(here)

    Location:android/device/*vendor*/*device*/

  2. Kernel tree

    3. This contains the Linux kernel source. It is setup according to the device's chipset config. That is, devices with same chipset can use the same kernel source!

    Location:android/*vendor*/*device*/ or android/*vendor*/*chipset*/

  3. Vendor tree

    4. This contains the library files needed my Android OS. These lib files(.so) are located in /system/lib & /system/vendor/lib on our phone running Android.

    Location:android/vendor/*vendor*/*device*/
